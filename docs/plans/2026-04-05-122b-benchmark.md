## Task 1: Write benchmark script

**File:** `benchmarks/model_benchmark.py`

Create a generic model benchmark script that:
- Accepts model path, prompt file, and output TSV path
- Uses mlx-lm for standard MLX models (122B)
- Uses flash-moe binary for flash-moe models (397B)
- Measures: generation tok/s, prompt tok/s, TTFT, peak Metal memory
- Runs each prompt 2x (warm-up + measured run)
